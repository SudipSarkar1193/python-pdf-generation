Testing Concepts

  • Chief objective of test team: To find out errors.
  • Error: It is the difference between actual output and theoretical output.
  • Failure: It is the inability of software system to produce error-free result.
  • Fault: It is known as algorithmic error, which error will come.

Fault → Error → Failure

V & V: Verification and Validation

  • Verification: Verification is the process by which software engineers do correctly with respect to the previous stage.
  • Validation: It is the process which is done at the end of testing phase and it checks the validity of the final product with respect to SRS document.

Test Suite: A set of test cases.
